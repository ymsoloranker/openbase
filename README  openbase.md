# 🚀 OpenBase

**Your workspace. Your machine. Your rules.**

OpenBase is a fully local, subscription-free Notion alternative — docs, wikis, notes, and task databases, all living on your own disk instead of someone else's server.

No login walls. No monthly bill. No "upgrade to unlock." Just a workspace that's actually yours.

---

## Why this exists

Every "productivity tool" eventually turns into a subscription trap or a cloud dependency you didn't ask for. OpenBase is the opposite bet: build the thing you'd actually want to use for the next 10 years, without worrying it gets paywalled, sunset, or held hostage to an internet connection.

Nobody has to be watching for the work to matter. This gets built either way.

---

## ✨ Features

- **New Page** — blank canvas, zero friction
- **Templates** — spin up pages fast without reinventing structure
- **Wiki** — a real knowledge base, not just a folder of docs
- **Team Wiki / Meeting Notes / Task Database** — the core pages every workspace actually needs
- **Favorites & Recent** — jump back into what matters, not what's alphabetically first
- **Quick Find** (`⌘K`) — search without leaving the keyboard
- **100% Local** — your data stays on your machine, no external database required
- **No subscription, ever**

## 📦 Getting Started

```bash
git clone https://github.com/ymsoloranker/openbase.git
cd openbase
npm install
npm run dev
```

Open it up, hit **New Page**, and start building your workspace.
